# simple-ui-flask

A simple Flask web app with a login page, home dashboard, and logs page.

## Run the app

1. Install dependencies:
   ```bash
   python3 -m pip install flask
   ```

2. Start the app:
   ```bash
   python3 app.py
   ```

3. Open the app in your browser:
   ```text
   http://127.0.0.1:5001/login
   ```
